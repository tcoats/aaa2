# aaa2
## Random and reasonably unique (and fairly adorable) human readable ids

## Usage
```javascript
var aaa2 = require('aaa2');

console.log(aaa2());
// "supercurious-senior-woodlouse"

console.log(aaa2(5));
// "unquiet-calm-omniscient-ornate-industrious-deer"
```
